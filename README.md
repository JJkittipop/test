# Smart Bag Project

## Description
Smart Bag คือกระเป๋าอัจฉริยะที่ช่วยแจ้งเตือนเมื่อผู้ใช้ลืมสิ่งของ
โดยใช้ ESP32, Oled GPS เเละBuzzer

## Installation
- ESP32
- Arduino IDE
- Library: TinyGPS++,  WiFi,

## Usage
1. เปิดอุปกรณ์
2. เชื่อมต่อผ่านแอป
3. ระบบจะแจ้งเตือนเมื่อสิ่งของหายไปจากกระเป๋า

## Features
- แจ้งเตือนผ่านเสียง Buzzer
- ส่งตำแหน่ง GPS
- เชื่อมต่อ Bluetooth

## Configuration
- ตั้งค่า WiFi
- ตั้งค่า Firebase URL
